## Sobre
Este projeto tem como objetivo demonstrar visualmente o funcionamento de algoritmos de ordenação utilizando Python e Streamlit. O usuário pode desenhar barras verticais em um canvas interativo, que representam os valores de um vetor, e então escolher diferentes algoritmos para visualizar o processo de ordenação passo a passo.

Entre os algoritmos implementados estão Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort, Shell Sort e Radix Sort. Durante a execução, o sistema atualiza graficamente cada etapa da ordenação, permitindo uma melhor compreensão do comportamento e da lógica de cada algoritmo.



## Pré-requisitos
- requeriments.txt
- Terminal (PowerShell no Windows)

## Instalação e como rodar
1. Clone ou baixe o repositorio
2. Entre na pasta do projeto
3. No terminal: pip install -r requirements.txt
4. Para rodar: python -m streamlit run app.py
5. Desenhe retângulos verticais 
6. Escolha o algoritmo desejado
7. Clique em "Extrair barras"
8. Clique em "Iniciar Ordenação"
